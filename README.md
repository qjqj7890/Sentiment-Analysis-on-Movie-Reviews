# Sentiment-Analysis-on-Movie-Reviews
English movie review sentiment analysis using RNN

## Key Achievements
- **Final Score:** 0.64033
- **Class Ranking:** Achieved a top-tier rank in the Deep Learning course at University of Seoul (UOS).
- **Project Constraints & Implementation:**
  1. **Strictly Scratch-built:** All architectures were implemented from scratch without using high-level libraries for model building.
  2. **Framework:** Developed exclusively using **PyTorch**.
  3. **No Pre-trained Models:** Successfully achieved high performance without relying on any pre-trained weights or models (e.g., BERT, RoBERTa).
  4. **Embedding:** Utilized **GloVe** as the sole word embedding method.

## Project Structure
- `technical_report.pdf`: Detailed technical report and methodology.
- `submission.csv`: Final prediction results with the best-performing model.
- `movie_classification.ipynb`: Complete implementation of the data pipeline and model training.
